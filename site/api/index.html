API guide

Main paths to learn first.

New code should prefer the compact public path. Lower-level explicit source APIs remain available for compatibility, tests, and diagnostics.

source_err(reason, detail)

First entry into structured errors, or a new semantic boundary that keeps the upstream error as source.

conv_err()

Reason-only conversion between layers. Use it when you do not want a new source boundary.

to_err()

Turn one reason value into StructError<R> directly.

with_source(...)

Attach a source through automatic routing, whether it is a raw standard error or another structured error.
